Ajuda?

Instalação do Ubuntu Server 20.04, tutorial simples, sem enrolação.

Clique no botão acima para fazer o download do Ubuntu Server. O download iniciará automáticamente.

Clique no botão acima para abrir o passo a passo de instalação da Máquima Virtual no Hiper-V.

Passo a Passo - Criando Máquima Virtual

1º Passo - Leia com atenção a caixa de informações iniciais e clique no botão avançar, para prosseguir.

2º Passo - Especificar o nome e versão desejada, em seguida localize local da sua ISO. para prosseguir.

3º Passo - Especifique a geração 1 ou geração 2. Clique no botão avançar, para prosseguir.

4º Passo - Atribuia mais memória, 2048 MB e clique no botão avançar, para prosseguir.

5º Passo - Altere de "Conexão: Não Conectado", para "Conexão: Default Switch" e clique no botão avançar, para prosseguir e observe como deverá ficar no passo 6º

6º Passo - Se Imagem estiver correspondente a sua instalação, clique no botão avançar, para prosseguir ou senão no botão anterior para voltar.

7º Passo - Ideal é alterar tamanho do disco para 30GB e clique no botão avançar, para prosseguir.

8º Passo - Opções de instalação, selecione "Inslatar um sistema operacional a parti de um CD/DVD-ROM inicializável", em seguida selecione a opção "Arquivo de Imagem (.iso) e em procurar. A partir de agora você irá localizar seu diretório, aonde deverá estar salvo seu "arquivo.iso", clique em OK e clique no botão avançar, para prosseguir.

Tela de conclusão.

Clique no botão acima para abrir o passo a passo de instalação do Ubuntu Server..

Passo a Passo - Instalando Ubuntu Server

1º Passo - Selecione o idioma desejado e clique enter, para prosseguir.

2º Passo - Selecione a opção desejada e clique enter, para prosseguir.

3º Passo - Selecione a opção desejada, se estiver com a versão atual, "Continue without updating" e clique enter, para prosseguir.

4º Passo - Selecione o Idioma com tecla "tab", veja imagem do 5º passo e clique enter, para prosseguir.

5º Passo - Selecione o Idioma com tecla desejado e clique enter, para prosseguir.

6º Selecionado Idioma, clique na tecla "tab" até "done" e clique enter, para prosseguir.

7º Configurando IP manual, clique em "enpOs3 eth" e selecione "Edit IPv4", e clique enter, para prosseguir.

8º Passo - Nesta janela "Edit enpOs3 Ipv4 configuration", opção "Manual" e clique enter, para prosseguir.

9º Passo - Configure seu "Ipv4 Method[manual]" conforme sua necessidade, "tab" para muda de janela e até "Save", para prosseguir.

10º Passo - Neste passo, não configuraremos proxy. Como a tecla "tab" até "Done" e clique enter, para prosseguir.

11º Passo - Neste passo, Como a tecla "tab" até "Done" e clique enter, para prosseguir.

12º Passo - Neste passo, deixa as configurações como estão. Como a tecla "tab" até "Done" e clique enter, para prosseguir.

13º Passo - Neste passo, Como a tecla "tab" até "Done" e clique enter, para prosseguir.

14º Passo - Como a tecla "tab" altere para "Continue" e clique enter, para prosseguir.

15º Passo - Deseja Instalar "OpenSSH server"? Se sim, aperte a tecla de "espaço" para selecionar "Install OpenSSH server". Se não, Como a tecla "tab" até "Done" e clique enter, para prosseguir.

16º Passo - Neste passo, não iremos instalar nenhum outro programa da lista abaixa. Como a tecla "tab" até "Done" e clique enter, para prosseguir.

Tela de instalação.

Tela de conclusão.

Intalação Concluída com Sucesso. Como a tecla "tab" até "Reboot Now" e clique enter, para reiniciar.
