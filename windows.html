Instalação do Windows Server 2012 R2, tutorial simples.

Clique no botão acima para acessar página de download Windows Server 2012.

Clique no botão acima para abrir o passo a passo de instalação da Máquima Virtual no Hiper-V.

Passo a Passo - Criando Máquima Virtual

1º Passo - Leia com atenção a caixa de informações iniciais e clique no botão avançar, para prosseguir.

2º Passo - Especificar o nome e versão desejada, em seguida localize local da sua ISO. para prosseguir.

3º Passo - Especifique a geração 1 ou geração 2. Clique no botão avançar, para prosseguir.

4º Passo - Atribuia mais memória, 2048 MB e clique no botão avançar, para prosseguir.

5º Passo - Altere de "Conexão: Não Conectado", para "Conexão: Default Switch" e clique no botão avançar, para prosseguir e observe como deverá ficar no passo 6º

6º Passo - Se Imagem estiver correspondente a sua instalação, clique no botão avançar, para prosseguir ou senão no botão anterior para voltar.

7º Passo - Ideal é alterar tamanho do disco para 30GB e clique no botão avançar, para prosseguir.

8º Passo - Opções de instalação, selecione "Inslatar um sistema operacional a parti de um CD/DVD-ROM inicializável", em seguida selecione a opção "Arquivo de Imagem (.iso) e em procurar. A partir de agora você irá localizar seu diretório, aonde deverá estar salvo seu "arquivo.iso", clique em OK e clique no botão avançar, para prosseguir.
